# A1 — Modern Angular Control Flow & Deferred Views

**Level:** Architect
**Skills tested:** Angular 17+ built-in control flow (`@if`, `@for`, `@switch`, `@empty`), `@defer` with trigger types, SSR awareness, hydration
**Files to edit:** All component templates + `src/app.component.html`

---

## Context

Angular 17 introduced a new built-in control flow syntax that replaces structural directives (`*ngIf`, `*ngFor`, `*ngSwitch`) and adds a powerful `@defer` block for lazy rendering. This challenge migrates the app to the new syntax and uses `@defer` to defer loading the Order Tracker until the user actually wants it.

## Tasks

### Task 1 — Migrate `*ngIf` to `@if`
Replace all `*ngIf` with the built-in `@if / @else` syntax.

**Before:**
```html
<app-menu *ngIf="showMenu"></app-menu>
<app-reservation-form *ngIf="showReservation"></app-reservation-form>
```

**After:**
```html
@if (showMenu) {
  <app-menu />
} @else if (showReservation) {
  <app-reservation-form />
} @else {
  <p>Select a view from the menu above.</p>
}
```

Also migrate all `*ngIf` in the reservation form's error messages.

### Task 2 — Migrate `*ngFor` to `@for` with `@empty`
Replace all `*ngFor` directives in the menu template:

```html
<!-- Before: -->
<mat-card *ngFor="let item of menuItems; trackBy: trackById">

<!-- After: -->
@for (item of menuItems; track item.id) {
  <mat-card class="menu-item">
    ...
  </mat-card>
} @empty {
  <p class="empty-state">No menu items available. Please check back later.</p>
}
```

### Task 3 — Add `@switch` for order status display

Use `MatIconModule` icons: `restaurant`, `outdoor_grill`, `delivery_dining`, `check_circle`.

### Task 4 — Wrap Order Tracker in `@defer`
In `app.component.html`, replace the `@if (showTracker)` block with a `@defer` block:

```html
<!-- TODO: Defer the OrderTrackerComponent so it only loads when the user
     clicks the "Show Order Tracker" button (on interaction trigger) -->

@defer (on interaction(trackerButton)) {
  <app-order-tracker />
} @placeholder {
  <!-- Skeleton: show a greyed-out box the same size as the tracker -->
  <div class="tracker-skeleton"></div>
} @loading (minimum 500ms) {
  <mat-progress-bar mode="indeterminate" />
} @error {
  <p>Failed to load order tracker.</p>
}
```

Add `#trackerButton` template reference to the "Show Order Tracker" button.

## Acceptance criteria

- [ ] No `*ngIf`, `*ngFor`, or `*ngSwitch` remain in any template
