## Context

Navigation is currently controlled by three boolean flags in `AppComponent`:
```typescript
showMenu = true;
showReservation = false;
showTracker = false;
```

This means:
- The URL never changes — you can't bookmark or share a view
- The browser back button does nothing
- All three components are eagerly loaded at startup even if never visited
- There is no mechanism to prevent navigating to the Order Tracker before an order exists

Your task is to replace this with a proper Angular Router setup.

---

## Current code

**`src/main.ts`:**
```typescript
import { bootstrapApplication } from '@angular/platform-browser';
import { AppComponent } from './app/app.component';
import { provideAnimations } from '@angular/platform-browser/animations';

bootstrapApplication(AppComponent, {
  providers: [provideAnimations()],
});
```

**`src/app.component.ts`:**
```typescript
export class AppComponent {
  showMenu = true;
  showReservation = false;
  showTracker = false;

  onToggleChange(event: any) {
    this.showTracker = event.checked;
  }
}
```
