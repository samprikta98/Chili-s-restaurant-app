## Context

The current order tracker has three problems:

1. **Memory leak** — `OrderTrackerComponent` subscribes to `orderStatus$` in `ngOnInit` but never unsubscribes. If the component is destroyed (user toggles the slide toggle off), the subscription keeps running.

2. **No error handling** — if `OrderTrackingService` throws (e.g. a real API fails), the observable terminates silently and the UI freezes on the last known status.

3. **Jittery UI** — the service emits a new random status every 5 seconds, including duplicates. The component re-renders even when the status hasn't changed.

---

## Current code

**`order-tracking.service.ts`:**
```typescript
@Injectable({ providedIn: 'root' })
export class OrderTrackingService {
  private orderStatus = new BehaviorSubject<string>('Not started');
  orderStatus$ = this.orderStatus.asObservable();
  private trackingSubscription: Subscription | undefined;

  statuses = ['Preparing', 'Cooking', 'Quality Check', 'Out for Delivery', 'Delivered'];

  startTracking() {
    this.stopTracking();
    this.trackingSubscription = interval(5000).subscribe(() => {
      const randomStatus = this.statuses[Math.floor(Math.random() * this.statuses.length)];
      this.orderStatus.next(randomStatus);
    });
  }

  stopTracking() {
    if (this.trackingSubscription) {
      this.trackingSubscription.unsubscribe();
    }
  }
}
```

**`order-tracker.component.ts`:**
```typescript
@Component({ ... })
export class OrderTrackerComponent implements OnInit {
  currentStatus: string = 'Not started';
  private statusSubscription: Subscription | undefined;

  constructor(private orderTrackingService: OrderTrackingService) {}

  ngOnInit() {
    this.statusSubscription = this.orderTrackingService.orderStatus$.subscribe(
      (status) => (this.currentStatus = status)
    );
    // BUG: no ngOnDestroy — statusSubscription is never cleaned up
  }

  startTracking() {
    this.orderTrackingService.startTracking();
  }
}
```

## Final pipe shape (in the component)

```typescript
this.orderTrackingService.orderStatus$.pipe(
  debounceTime(300),
  retry(2),
  catchError(err => { /* set error state; return EMPTY */ }),
  takeUntilDestroyed(this.destroyRef)
).subscribe(status => this.currentStatus = status);
```
