# Challenge 2: Advanced Angular Scenarios

These scenarios test intermediate-to-advanced Angular concepts in real-world restaurant app contexts.

## Prerequisites
- Familiarity with RxJS, Angular Signals, and reactive patterns

## Scenarios

| # | Topic | Difficulty | Time |
|---|-------|------------|------|
| 1 | Implementation of Cart service 
| 2 | Performance Optimization 
| 3 | Unit Testing for the cart service implemented in scenario 1
| 4 | Architecture Refactoring
| 5 | Debugging UI
| 6 | Menu search functionality using RxJs
| 7 | Order Customization form
| 8 | Zoneless Angular debugging
